# Atlantis
Atlantis an ecosystem model that considers all parts of marine ecosystems – biophysical, economic, and social. Originally focused on the biophysical world and then on fisheries, it has grown to be used for cumulative impacts and climate-related questions.
#### https://research.csiro.au/atlantis/home/about-atlantis/
Developed by Dr. Beth Fulton at the Commonwealth Scientific and Industrial Research Organization
